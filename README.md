# Rust By Templates
This repository serves the purpose of providing a collection of Rust templates for quickly building applications for new Rust developers.

## How To Use
Each template has a README with instructions on usage.


## Features
- [ ] HTTP server
	- [ ] Simple HTTP server with axum.
